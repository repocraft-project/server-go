# AGENTS.md

## Project Goal

This project is a Git hosting platform similar to GitHub and Gitea, built with Go on the server side.

## Development Guidelines

1. **Write code only**: Do not manually run programs for testing to avoid impacting the project
2. **Code style**:
   - Do not add any comments unless explicitly requested
   - Follow Go language conventions
3. **After completing tasks**: Run lint and typecheck commands to ensure code correctness
4. **Do not commit proactively**: Unless explicitly requested by the user

## Project Structure

- `cmd/` - Application entry points
- `internal/` - Internal packages
